# Vextura Releases

Public release artifacts for the Vextura Workflow Engine.

- **vex CLI** — `vex_*` archives for Linux, macOS, Windows
- **Java 21 SDK** — `vextura-sdk-*.jar`
- **Java 8 SDK** — `vextura-sdk-java8-*.jar`
- **Engine compose** — `docker-compose.client-*.yml`

## Install vex CLI

```sh
curl -sSL https://github.com/vextura/releases/releases/latest/download/install.sh | sh
```

## TypeScript SDK

```sh
npm install @vextura/sdk
```
